Detailed Description

This is a wrapper class for a Gtk::Dialog. It will generate a dialog for the user to select the desired processes for which a breakpoint should be set or deleted. This information is saved in the m_button_states array.

This is the default constructor for the BreakpointDialog class. This will generate a grid of checkbuttons that are, if the init flag is set, all checked. Otherwise the state of the checkbuttons depends on the state of the breakpoint at the corresponding process.

◆ on_dialog_response()

void BreakpointDialog::on_dialog_response ( const int  response_id)
private

This function gets called when the user clicks on the "Ok" button or the close button. It populates the m_button_states array with the state of each checkbutton. This array will be used by the Breakpoint class to set/delete breakpoints per process.
